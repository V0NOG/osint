A — Markers + Event Arcs (Additive)

Keep the current dot-marker approach. Phase 4 adds: animated event arcs between related countries, improved hover glow on markers, and a richer country tooltip. No GeoJSON needed — fast, focused, done in one phase.

B — Country Boundaries on Globe

Load GeoJSON world data and render country outlines as polygons on the sphere surface, color-filled by risk level. Countries highlight on hover. Dot markers remain for high/critical pulse. Cinematic, intelligence-terminal feel — looks like a real geospatial tool.

Uses map texture + risk overlay

C — Texture + Risk Overlay

Apply a dark world map texture to the sphere, then overlay a second risk-colored texture (generated from country risk scores). Visually rich with minimal Three.js complexity. Less interactive than polygon approach — can't click individual country shapes.
